## Описание программы

** Random Task Generator ** - это GUI-приложение предназначено для генерации случайных задач по разным направлениям (учеба, работа, спорт) из заранее заданного списка и ведения истории сгенерированных заданий. Предусмотрено добавление задачи самостоятельно.

### Возможности:
- ✅ Генерация случайных задач из списка
- ✅ Добавление своих задач
- ✅  Сохранение в JSON 
- ✅  Фильтрация задач всех



## Инструкция по запуску

### Требования:
- Python 3.6 или выше
- Установленная библиотека Tkinter и Random
-  Сохранение в JSON

### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
3. Открыть файл  main.py
4.Запустить приложение python main.py
   
## Примеры использования
1.Список задач — в начале файла задан список задач с их типами.
2.Кнопка «Сгенерировать задачу» — при нажатии случайная задача добавляется в историю и сохраняется в файл tasks.json.
3.История — отображается в виде списка внизу окна.
4.Фильтр — можно выбрать тип задачи («все», «учёба», «спорт», «работа»), чтобы видеть только нужные.
5.Сохранение — история сохраняется в tasks.json.и загружается при запуске программы.

### Пример 1: Генерация задачи
- Нажмите кнопку «Сгенерировать задачу»
- Появится случайная задача, например: «Сделать зарядку 15 минут»
- Задача добавится в историю снизу
